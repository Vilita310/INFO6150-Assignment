# Assignment 8 
INFO 6150 

## Overview
Lists books (GET), shows book detail (GET), and creates a new book (POST) using API:
`https://691688f8a7a34288a27d98c0.mockapi.io/books`

Fields: title (required), author (required), publisher, year, pages, topics.

## Files
- `index.html` – list + search + “Create” link  
- `detail.html` – one item detail (by `id`, fallback by `title`)  
- `create.html` – form to POST a new item (basic validation)  
- `styles.css` – simple, readable CSS  
- `app.js` – all JS (fetch + render + validation + tiny router)

## Run
Put all files in the same folder and open `index.html` in the browser.
